Gmail Gerrit Extension Setup

When you're viewing an email from Gerrit,

  • This extension will format your emails more nicely.
  • You can press 'w' to open the review diffs page.

Settings

Gerrit URL:
The URL where your Gerrit instance is hosted.
Enabled Gmail Account:
The Gmail address that you want to enable the Gerrit plugin for. If left empty, enable for all addresses.

Display

Context Lines:
Number of context lines to display when showing diffs.
Bot names:
Comma-separated list of usernames of automated bots that also comment on your Gerrit changes (like jenkins, etc.)
